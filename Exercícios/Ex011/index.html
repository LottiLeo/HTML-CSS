Trabalhando com Links

Quais são as tags utilizadas

Para inserir um link utilizamos a tag <a> que significa âncora, dentro dessa tag utilizamos href.
O resultado fica: <a href="link">
</a>

HREFLANG

Para indicar o idioma principal do site que será aberto, utiliza-se a tag hreflang.
Desse modo, a tag completa seria: <a href="link" hreflang="en">
</a>

Explicando na prática

Você pode acessar o meu perfil no Linkedin desse modo

Para abri o mesmo site em outra aba, após o href utilizamos o comando targets="_blank"depois do link

Utilizando Links internos

Utilizamos links internos para nos movermos entre um arquivo e outro dentro do nosso servidor e diz respeito a nossa própria página.
Voê pode acessar a segunda página desse documento clicando aqui
Você também pode acessar um link que está em outra pasta, desde que ao inseriro link no href consiga descrever o caminho deste documento. Por exemplo: Você pode acessar nossa página de notícias

Vamos fazer um link de download

Para fazer um link de download utilizo os mesmos comandos, especificando o caminho, acrescentando o atributo "download" e exemplificando o tipo do arquivo a ser baixado com o comando "type"(text/video,zip,image)
