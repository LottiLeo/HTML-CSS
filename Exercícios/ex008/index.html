Principais Formatações

Negrito/Destaque

Utilizando o comando b podemos deixar a frase em destaque

Itálico

Utilizando o comando em podemos deixar a frase em itálico

Marca Texto

Utilizando o comando mark podemos deixar a frase marcada

Sublinhado

Utilizando o comando u podemos deixar a frase sublinhada

Texto Excluído

Utilizando o comando del podemos deixar um texto excluído

Texto Inserido

Utilizando o comando ins podemos deixar um texto Inserido

Texto Sobrescrito

Utilizando o comando sup podemos deixar o texto sobrescrito como por exemplo na porência a seguir x20*3

Texto Subescrito

Utilizando o comando sub podemos deixar o texto subescrito como por exemplo na fórmula da água H2O
